## 主要功能
### 订阅页面

* 下载Clash订阅
* 删除订阅
* 切换订阅
* 批量更新订阅
* 从返回头自动获取配置名称和流量信息


### 控制页面

* 启动核心
* 停止核心
* 打开WebUI
* 查看核心状态

## 食用方法
* 前往Release下载对应架构的APK和mihomo.zip
* 安装APK并授予Root权限
* 添加订阅并启动核心
* 前往WebUI观察运行情况




## 编译

* 本项目采用GithubAction进行编译
* Fork或Import本项目,在Action选项卡内运行Build and Release Flutter APK
* 等待成功运行完成后在Action产物里寻找并下载
